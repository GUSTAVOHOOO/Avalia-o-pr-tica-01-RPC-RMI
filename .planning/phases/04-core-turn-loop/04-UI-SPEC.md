# Phase 4 — UI Design Contract

> Visual and interaction contract for Phase 4: Core Turn Loop.
> Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none (Vite + React + TypeScript, no shadcn) |
| Preset | not applicable |
| Component library | none (inline styles + Tailwind utilities, matching Phase 3 pattern) |
| Icon library | lucide-react ^0.511.0 (already in package.json) |

## Interaction States

### Hint input lifecycle
1. HINT_PHASE starts → input enabled, button enabled
2. Player types → button remains disabled if input is empty (trim check)
3. Player submits → optimistic UI: input disabled, button becomes "Dica enviada" with #6b7280 text and #1a1d27 background
4. Counter updates on each `hint_received` event (aria-live="polite")
5. GUESS_PHASE starts → HintPhasePanel is replaced by GuessPhasePanel

### Guess input lifecycle
1. GUESS_PHASE starts → all controls enabled; hints chips populated from `PHASE_CHANGED` payload hints dict
2. Player must select target AND type guess before submit button activates
3. Player submits → optimistic UI: all controls disabled, feedback row shows "Aguardando resultado..."
4. `GUESS_RESULT` arrives → feedback row updates to "Correto!" or "Errado."
5. Player skips → "Passar" clicked: all controls disabled, submit button shows "Palpite enviado", no feedback row shown
6. SCORING_PHASE starts → GuessPhasePanel is replaced by ScoringPhasePanel

### Scoring panel lifecycle
1. SCORING_PHASE starts → ScoringPhasePanel appears with "Calculando pontuação..." state
2. `SCORE_UPDATED` arrives → rows populate from `scores` array in payload
3. TURN_END → ScoringPhasePanel replaced by passive text
4. New HINT_PHASE (next turn) → SecretImagePanel refreshes with new image URL from next `OBJECT_ASSIGNED`

## Accessibility

- All interactive controls have accessible labels via visible `<label>` elements or `aria-label` attributes
- Hint counter uses `aria-live="polite"` so screen reader announces updates without interrupting
- Guess result feedback uses `aria-live="polite"`
- Disabled buttons use both `disabled` attribute and `aria-disabled="true"` (matches Lobby.tsx pattern)
- Secret image uses `alt` attribute: `alt="Imagem do objeto: {object_name}"`
- Color is never the sole differentiator — turn delta sign (+ / −) supplements color, score labels supplement color
