<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser → Flask-SocketIO | WebSocket connection; browsers treated as untrusted callers |
| BridgeCallbackReceiver → socketio.emit() | Cross-thread call from Pyro5 daemon thread to SocketIO; safe in threading mode |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-09 | Elevation of Privilege | bridge/bridge.py Flask bind | mitigate | socketio.run(host="127.0.0.1") — loopback only in Phase 1; never 0.0.0.0 |
| T-01-10 | Information Disclosure | Flask SECRET_KEY = "dev-secret" | accept | Localhost Phase 1 only; acceptable per security context. Log a warning if bridge starts with non-localhost host. |
| T-01-11 | Tampering | get_game_server_proxy() per-thread proxy | accept | Per D-10, threading.local() isolates proxies; no shared state; Phase 1 callers are trusted local processes |
| T-01-12 | Denial of Service | Bridge startup retry loop | accept | 10-second timeout with clear error message; developer explicitly restarts if NS/GameServer not running |
</threat_model>
